- 상황:  
  교수님이 중간고사 문제 대비로 작년 기출문제를 알려주셨고 이에 대한 솔루션을 시험 전에 미리 작성. 시험 당일 날 **기출문제에서 수정된 변수 범위를 제대로 검사하지 못해, 추가 요구사항까지 반영한 코드를 완성했었으나 인자값을 잘못 전달해 추가점수는 물론이고 기본점수에서도 까임**
